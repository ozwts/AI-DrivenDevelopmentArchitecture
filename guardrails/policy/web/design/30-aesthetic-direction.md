# 美的方向性

## 核心原則

4原則（構造）に加え、プロジェクトごとに明確な**美的方向性（トーン）**を選択し、トークン体系に一貫して反映する。

**根拠となる憲法**:
- `design-principles.md`: 美的方向性の反映

## トーン選択のプロセス

実装前に以下を検討する：

| 観点 | 検討内容 |
|------|---------|
| 目的 | このインターフェースが解決する問題は何か |
| 利用者 | 誰が使うのか、どのような文脈で使うのか |
| トーン | どのような美的方向性が適切か |
| 差別化 | 記憶に残る独自の特徴は何か |

## トーン別トークン設計の指針

| トーン | フォント | 配色 | Spacing | 特徴 |
|--------|---------|------|---------|------|
| ミニマル | サンセリフ、細め | モノクロ+1アクセント | 広め | 余白を活かした静謐さ |
| レトロ | セリフ、クラシック | 暖色系、セピア | 中程度 | 懐かしさ、温かみ |
| ブルータリスト | 太め、幾何学的 | 高コントラスト | タイト | 大胆さ、力強さ |
| 豪華・洗練 | エレガントなセリフ | ゴールド、深い色 | ゆったり | 高級感、信頼感 |
| 遊び心 | 丸み、手書き風 | 鮮やか、多色 | 変化あり | 親しみやすさ |

## 表現要素の設計

### タイポグラフィ

| 要素 | 指針 |
|------|------|
| フォントペアリング | 見出し用と本文用を意図的に組み合わせる |
| ディスプレイフォント | 見出し・ヒーローに個性的なフォントを検討 |
| 本文フォント | 可読性を最優先、長文でも疲れない選択 |
| ウェイトの階層 | 2〜3段階に限定し、明確な対比を作る |

### モーション・アニメーション

| 場面 | 指針 |
|------|------|
| ページロード | 控えめなフェードイン、スタガー効果 |
| ホバー・インタラクション | 即座のフィードバック、微細な変化 |
| 状態遷移 | 自然な緩急（ease-out推奨） |
| スクロール連動 | 過度な視差効果は避ける |

**原則**: モーションは装飾ではなく、状態変化の明示。過剰なアニメーションは認知負荷を増やす。

### 空間構成

| 手法 | 効果 | 注意点 |
|------|------|--------|
| 非対称レイアウト | 視覚的な興味、動的な印象 | 整列原則との両立が必要 |
| オーバーラップ | 奥行き、関連性の強調 | 情報の優先度を損なわない |
| グリッドブレイク | アクセント、注目の誘導 | 意図的な逸脱のみ許容 |

### 背景・装飾

| 要素 | 指針 |
|------|------|
| グラデーション | 控えめに使用、トーンに合った色選択 |
| テクスチャ | 質感の付与、ただし可読性を損なわない |
| 幾何学パターン | ブランドアイデンティティの表現 |
| イラスト・アイコン | 統一されたスタイル、トーンとの整合 |

## 避けるべきもの

**ジェネリックなAI生成デザイン**:
- グラデーション多用（特に紫系）
- 汎用フォントの無個性な使用（Arial、Inter、Roboto等）
- 定型的なレイアウトやコンポーネント
- 文脈に合わない汎用的デザイン
- 過剰なアニメーション・視差効果

## 本プロジェクトの美的方向性

### 選択したトーン: 洗練されたミニマル

| 要素 | 選択 | 理由 |
|------|------|------|
| フォント | Noto Sans JP | 可読性と日本語対応 |
| 配色 | ライトベージュ + ダークブルー | 温かみと信頼感の両立 |
| Spacing | 広め | 余白を活かした静謐さ |
| 角丸 | 控えめ（sm〜md） | 柔らかさを維持しつつ洗練 |

### トークンへの反映

```javascript
// tailwind.config.js での表現
colors: {
  primary: { DEFAULT: "#f0ebe3" },     // ライトベージュ - 温かみ
  secondary: { DEFAULT: "#001964" },   // ダークブルー - 信頼感
  background: { DEFAULT: "#f0ebe3" },  // ベージュ背景 - 静謐さ
  text: {
    primary: "#000000",                // 高コントラスト
    secondary: "#333333",
    tertiary: "#7d6e5a",               // ベージュ系の補色
  },
}
```

### 差別化ポイント

- ベージュ背景による温かみ（一般的な白/グレー背景との差別化）
- ダークブルーのアクセント（紫グラデーションの回避）
- 十分な余白による高級感
