# 用語集

| 用語 | 英語名 | 定義 |
|------|--------|------|
| ユーザー | User | アプリケーションを利用する人。認証済みのアカウントを持つ |
| TODO | Todo | やるべきタスク。タイトル、説明、ステータス、優先度、期限、担当者を持つ |
| TODOステータス | Todo Status | TODOの進捗状況。未着手、作業中、完了のいずれか |
| 優先度 | Priority | TODOの重要度。低、中、高のいずれか |
| 添付ファイル | Attachment | TODOに添付されるファイル。ファイル名、サイズ、種類を持つ |
| 添付ステータス | Attachment Status | 添付ファイルのアップロード状況。準備完了、アップロード完了のいずれか |
| プロジェクト | Project | TODOをグループ化するための分類。名前、説明、カラーを持つ |
| プロジェクトロール | Project Role | プロジェクト内での権限レベル。プロジェクトオーナー（管理権限：設定変更、メンバー管理、Todo全操作）、プロジェクトメンバー（Todo全操作のみ）のいずれか |
| プロジェクトメンバーシップ | Project Membership | ユーザーとプロジェクトの所属関係。プロジェクトロールを持つ |
| 担当者 | Assignee | TODOに割り当てられた責任者。プロジェクトメンバーシップを持つユーザーの一人 |
